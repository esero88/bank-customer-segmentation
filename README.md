# 🏦 Banking Customer Analytics — Kaggle Notebooks

This branch contains Kaggle notebooks and exploratory analytics workflows for the banking customer segmentation project.

The notebooks focus on:

- Data cleaning
- Feature engineering
- Exploratory data analysis
- Behavioral transaction analysis
- Customer segmentation preparation

---

## 📓 Available Notebooks

### 01_data_loading.ipynb
- Data ingestion
- Initial dataset exploration
- Schema inspection

### 02_data_cleaning.ipynb
- Missing value handling
- Date corrections
- Feature engineering
- Data standardization

### 03_feature_engineering.ipynb
- Customer enrichment
- Behavioral tagging
- Segmentation preparation

---

## 📊 Dataset

Bank Customer Segmentation Dataset from Kaggle:

- TransactionID
- CustomerID
- CustomerDOB
- Customer Location
- Account Balance
- Transaction Amount
